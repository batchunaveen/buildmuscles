# FORME Workout

An equipment-aware workout planner and interactive muscle explorer.

## Features

- Interactive front/back muscle map
- Equipment selector for home and commercial gym training
- Exercise matching by available equipment and muscle group
- Persistent workout builder using local storage
- Responsive mobile and desktop design
- Sample nutrition cards

## Run locally

No build tools are required. Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
